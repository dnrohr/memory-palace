# Golden Test Memory Set v0.1

Golden memories are platform-neutral fixtures for extraction, tagging, search, and export regression tests.

The canonical machine-readable set lives in `src/fixtures/goldenMemories.ts`.

## Current Cases

- Childhood pet memory with grade inference
- College roommate memory with explicit year
- First job memory with place and activity tags
- Moving apartment memory with approximate place tags
- Vacation memory with family and place tags
- Grief memory with emotion and pet tags
- Ambiguous date memory
- Multiple people with the same name
